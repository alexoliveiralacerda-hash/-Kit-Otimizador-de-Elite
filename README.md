🚀 Otimizador de Elite V7.0 (Supreme Hybrid)
O Otimizador de Elite é uma ferramenta avançada de manutenção e otimização de sistema desenvolvida em Python e PowerShell. Foi projetada para reduzir a latência, eliminar arquivos desnecessários e maximizar a performance do Windows, seja para trabalho multitarefa ou jogos competitivos.

🛠️ Funcionalidades Principais

Executar Otimização: Realiza uma limpeza profunda de caches, remove bloatware (aplicativos nativos desnecessários) e ajusta chaves de registro para alta performance.

Perfis de Prioridade (CPU):


Trabalho (Multitarefa): Prioridade equilibrada para fluidez em múltiplas aplicações (Win32PrioritySeparation 26).


Gaming (Foco FPS): Ajusta o sistema para dar foco total ao processo em primeiro plano, reduzindo o stuttering (Win32PrioritySeparation 38).


Agendar Manutenção: Cria uma tarefa automática no Agendador de Tarefas do Windows para executar a limpeza todos os domingos às 03:00.


Restaurar Padrão: Uma ferramenta de segurança que reverte as modificações de registro, rede e interface para os valores originais do sistema.


Monitoramento Real-time: Painel com estatísticas ao vivo de uso de CPU, memória RAM e espaço livre em disco.

📋 Requisitos de Sistema
Sistema Operativo: Windows 10 ou Windows 11 (64 bits).


Permissões: É obrigatório executar como Administrador para permitir alterações no Registro e limpeza de pastas do sistema.


Dependências: Caso execute via script, requer Python 3.x e as bibliotecas customtkinter, psutil e Pillow.

🚀 Como Utilizar
Acesse a aba Releases no GitHub e baixe o executável main.exe.

Clique com o botão direito no arquivo e selecione "Executar como Administrador".

Escolha o seu Perfil de Prioridade desejado.

Clique em "EXECUTAR OTIMIZAÇÃO" e acompanhe o progresso em tempo real.

Dica: Recomendamos reiniciar o computador após o término para que todas as alterações de hardware e rede entrem em vigor.

⚠️ Segurança e Transparência

Ponto de Restauro: O software tenta criar um ponto de restauração automaticamente antes de iniciar qualquer procedimento.

Código Aberto: Todo o processo é transparente. Você pode verificar os arquivos .ps1 para entender exatamente quais chaves de registro e pastas estão sendo modificadas.


Interrupção Forçada: O botão "PARAR PROCESSO" encerra imediatamente a execução dos scripts em segundo plano.

⚠️ Alerta de "Falso Positivo"
Como este software realiza alterações profundas no Registro do Windows e não possui uma assinatura digital paga, navegadores como o Chrome e o Windows SmartScreen podem emitir alertas.

Para baixar: No Chrome, clique na seta ao lado de "Descartar" e selecione "Manter assim mesmo".
Para executar: Clique em "Mais informações" e depois em "Executar assim mesmo".
